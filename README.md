# Mobile app created in CareerFoundry Full Stack Developer course
<ul>
<li>The App was built using React Native and developed using Expo.</li>
<li>Testing was done with both ios and Android emulators.</li>
<li>Styling met the given project Design Screens.</li>
<li>Chat conversations are stored in Google FireStore Database and locally.</li>
<li>The app allows users to pick & send images from the phone's image library.</li> 
<li>The app also lets users take pictures with the device’s camera app, and send them.</li>
<li>The app stores images in Firebase Cloud Storage.</li>
<li>The app is able to read the user’s location data</li>
<li>Location data can be sent via the chat in a map view.</li>
<li>The chat interface and functionality was created using the Gifted Chat library.</li>
<li>The app’s codebase includes comments where appropriate.</li>
</ul>

## TO RUN THE APP

1. Clone this repository 
2. Open the program files in an editor of your choice
3. run npm install
4. run Expo start (to download expo vist the app store on your device or use the web app)
